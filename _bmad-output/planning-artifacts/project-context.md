## What This Is

Rentify is a **Cambodia-first peer-to-peer rental marketplace backend** that enables renting items (cameras, bikes, power tools) and services (photographers, movers, repair technicians). Built with Hono + Supabase on Cloudflare Workers, it handles discovery, trust, booking, payments via ABA PayWay, messaging, and dispute resolution.

## Core Value

**Trust-infrastructure for peer-to-peer rentals in emerging markets.** If everything else fails, users must be able to transact safely with strangers.

## Tech Stack

| Concern | Choice | Why |
|---------|--------|-----|
| API Framework | Hono (on Bun) | Lightweight, TypeScript-native, edge-ready |
| Database | Supabase (PostgreSQL) | Auth, storage, realtime, RLS built-in |
| Auth | Supabase Auth | Email/password + OAuth, JWTs handled |
| Storage | Supabase Storage | Media uploads with CDN |
| Payments | ABA PayWay | Cambodia's leading payment gateway |
| Deployment | Cloudflare Workers | Edge deployment, low latency |
| Validation | Zod | Type-safe schema validation |
